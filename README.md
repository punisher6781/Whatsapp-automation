## 📱 Overview

A WhatsApp-style automation system that allows sending predefined messages to multiple clients automatically without manual typing.

Designed especially for service centers to send alerts, awareness messages, and notifications efficiently.

---

## ✨ Features

- 📤 Bulk message sending system  
- 🧾 Predefined message (auto-fetched from backend)  
- 💬 WhatsApp-like UI interface  
- ✔✔ Message delivery ticks simulation  
- ⚡ Fast & lightweight system  
- 🔌 Ready for real WhatsApp API integration

Sends request to backend for each client
Backend processes message sending
UI displays message with ticks ✔✔
